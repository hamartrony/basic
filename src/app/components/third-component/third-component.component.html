Aula 3 - Diretivas Ng

Utilizadas para trazer funções automatizadas ao sistema.

Comecam com ng + nome da diretiva.

[ngClass] = insere classe no componente dinamicamente.

[ngStyle] = altera css do compoente.

[ngStyle]="{"font-size":size + "px"}"

*ngIf = formatação condicional; else outraCoisa

Sera mostrado se a Variavel SHOW for TRUE.

ng-template #outroElemento renderiza o que estiver em else

SHOW agora e FALSE
